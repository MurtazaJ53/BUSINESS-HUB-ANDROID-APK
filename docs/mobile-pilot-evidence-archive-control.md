# Mobile Pilot Evidence Archive Control

## Purpose

The mobile evidence tracker now supports two archive-level actions:

- copy the full evidence archive pack
- clear archived sessions while keeping the active session

This gives rollout leads a cleaner lifecycle on shared devices.

## Full archive pack

Use `Copy full archive pack` when you want:

- the active evidence tracker
- the recent archived session summaries
- one combined device-side export for rollout records

This is useful when the device has already been through multiple pilot shifts and the lead wants one combined handoff block.

## Clear archived sessions

Use `Clear archived sessions` when:

- the archived history is no longer useful
- the device is being repurposed for a new rollout wave
- you want to remove stale prior-session summaries without deleting the active session

Important:

- this does **not** reset the active session
- it only removes the archived session list

## Recommended use

1. Keep the active session while the current shift or wave is in progress.
2. Start a fresh session when a real operational boundary changes.
3. Use the archive pack when the rollout lead wants recent device history.
4. Clear archived sessions only when that recent history is no longer operationally useful.
